Function that controls possible loop in dataset reading.

HD5 dataset are often stored in groups: this function is needed to check that no dataset is read more than one time.
groupcheck returns 1 if the iterator goes through the same dataset two or more times. Returns 0 otherwise.

Parameters
loc_idLocation identifier of the dataset stored in memory
odPointer to the struct opdata used to store the information about group structure and its dataset organization
target_tokenToken of the currently opened dataset. It will be compared to other tokens stored in opdata
