Header file for all pragma directives needed by rootcling. 

All these directives are referred to R\+O\+OT internal compiler\+: rootcling. Note that these directives have to be correctly ordered.~\newline
 These commands are used to correctly write a dictionary to let a structure be writable to a tree.~\newline
The + which ends every pragma call let rootcling use the new I/O system of root.
