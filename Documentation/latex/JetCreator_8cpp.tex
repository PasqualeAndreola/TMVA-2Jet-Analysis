Function that clusters particles to jet. 

This function is quite useful for user purpose. \mbox{\hyperlink{JetCreator_8cpp_afc25fe7a7dda2fc0a6c65aa0eb0c5aac}{Jet\+Creator}} is designed to work with these settings\+:
\begin{DoxyItemize}
\item \char`\"{}anti-\/kt\char`\"{} algorithm, because hard jets are exactly circular on the $y-\phi$ cylinder
\item \char`\"{}\+E-\/scheme\char`\"{} it is the default option used by most experiment working with jets.~\newline
 It is designed to simply add the four vectors given as input to the clustering algorithm
\item \char`\"{}\+Best\char`\"{} strategy, which choose the optimal speed strategy for each multiplicity of jet
\end{DoxyItemize}
