Function that creates a $\ast$.root file containing signal/background/event kinematic quantities of the two most massive jets. 

\begin{DoxyAuthor}{Author}
Pasquale Andreola
\end{DoxyAuthor}
This function is designed to create as much trees as initial files.~\newline
These trees will store the kinematic quantities (px, py, pz, E) of the two most massive jets.~\newline
These jets are clustered using {\ttfamily R} as jet radius. The {\ttfamily p\+T\+\_\+min} is the minimum transverse momentum accepted by user analysis.~\newline
Jets are created by the \mbox{\hyperlink{JetCreator_8cpp_afc25fe7a7dda2fc0a6c65aa0eb0c5aac}{Jet\+Creator}} function, so please refer to its documentation to find the characteristics of the algorithm.~\newline
Returns 0 if there is no error during execution.
