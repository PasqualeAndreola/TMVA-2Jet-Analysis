This function is used to read a dataset of dimension 1 or 2 from a *.h5 file.

This function is used to read a dataset of dimension 1 or 2 from a *.h5 file.
Dataset_2Dread reads the dataset given by the dataset_id scanning it with memory slab whose dimensions are designed to be located inside dataset limit.
This is done because H5DRead is designed to crash if the user tries to read out of the dataset dimensions.
bufferdimension Determines the dimension of the buffer used to collect data from the *.h5 file.
It has to be chosen considering the datatype and the amount of RAM installed.
Memory slab read from the dataset is stored in a buffer bufferdimension bufferdimension (for 2D dataset or bufferdimension*1 for 1D dataset) of type buftype.
Buffer is then written in the "output".root file

Template Parameters
buftypeType of the buffer that is created to store data collected from the dataset
Parameters
dataset_idUnique dataset identifier, returned when the dataset is opened in TreeCreator
datasetnumberNumber assigned to the dataset when it is found from the openh5.
It is used to manage dataset information stored in the DatasetInfo structure
*outfileRepresents the name of the "output".root file
