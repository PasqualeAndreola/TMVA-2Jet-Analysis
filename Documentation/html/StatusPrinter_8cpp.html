void StatusPrinter ( int  progress)

Function used to print a banner with a progression bar.

This function is used to print a banner with some "[====>] x%" characters.
They contains information about the runtime and the progress of time consuming functions.
Global variable ncolumns can be defined by the user: the function will appropriately cut the progress bar.
