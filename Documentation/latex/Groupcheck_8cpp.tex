Function that controls possible loop in dataset reading. 

H\+D5 dataset are often stored in groups\+: this function is needed to check that no dataset is read more than one time.~\newline
\mbox{\hyperlink{Groupcheck_8h_ad5f2250dcc568f6d143b607a2aaed4b3}{groupcheck}} returns 1 if the iterator goes through the same dataset two or more times. Returns 0 otherwise.
