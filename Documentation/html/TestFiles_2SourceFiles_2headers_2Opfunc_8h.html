This is the function used as an iterative operator called by openh5 function.

While going through a *.h5 file, this function identifies info about every group/dataset/datatype, storing them in a list.
Returns a structure holding the error stack: it can be read using some HDF5 function. Refer to its documentation

Parameters
loc_id*.h5 file location identifier, obtained from openh5 function
*namePointer to the name of the *.h5 file
*infoStructure that holds info about the group/dataset/datatype read from the iteration
*operator_dataPointer to opdata structure, that will hold tokens of all the dataset in the file
