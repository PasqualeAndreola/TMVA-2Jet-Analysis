Function that clusters particles to jet.

This function is quite useful for user purpose. JetCreator is designed to work with these settings:

  • "anti-kt" algorithm, because hard jets are exactly circular on the \(y-\phi\) cylinder
  • "E-scheme" it is the default option used by most experiment working with jets.
    It is designed to simply add the four vectors given as input to the clustering algorithm
  • "Best" strategy, which choose the optimal speed strategy for each multiplicity of jet
Parameters
particlesList of particles forming the jets. A particle is defined, in this case, as a 4-vector
RJet radius, usually related to the angular reach of the jet. Can be found in the anti-kT algorithm formula
pT_minMinimum transverse jet momentum that is accepted for user's analysis
