Function that uses tmva methods to generate weight files for the relative algorithm.

Author
Pasquale Andreola

Some variables are added to a loader that will be passed to the factory. The user set up some tmva algorithms and the factory applies them to the sample dataset. Results are stored in some weight files.

Parameters
tmvamethodsVector of method names that user would use to analyze the sample
neventsNumber of training (signal/background) and test (signal/background) events that algorithms have to use
SBJetOutputFilenameName of the file which stores dijet kinematics
TMVAOutputFilenameName of the output file which will store factory results

< Number of events used to train signal identification in the sample

< Number of events used to train background identification in the sample

< Number of events used to test signal identification in the sample

< Number of events used to test background identification in the sample
