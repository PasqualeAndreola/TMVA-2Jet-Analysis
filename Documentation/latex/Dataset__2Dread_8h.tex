This function is used to read a dataset of dimension 1 or 2 from a $\ast$.h5 file.~\newline
\mbox{\hyperlink{Dataset__2Dread_8h_a2dc8bccd62c373531d819d9900038ebc}{Dataset\+\_\+2\+Dread}} reads the dataset given by the {\ttfamily dataset\+\_\+id} scanning it with memory slab whose dimensions are designed to be located inside dataset limit.~\newline
This is done because {\ttfamily H5\+D\+Read} is designed to crash if the user tries to read out of the dataset dimensions.~\newline
{\ttfamily bufferdimension} Determines the dimension of the buffer used to collect data from the $\ast$.h5 file.~\newline
It has to be chosen considering the datatype and the amount of R\+AM installed.~\newline
Memory slab read from the dataset is stored in a buffer {\ttfamily bufferdimension} {\ttfamily bufferdimension} (for 2D dataset or {\ttfamily bufferdimension$\ast$1} for 1D dataset) of type {\ttfamily buftype}. ~\newline
Buffer is then written in the \char`\"{}output\char`\"{}.root file
