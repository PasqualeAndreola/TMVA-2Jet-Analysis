Header file for \mbox{\hyperlink{Dataset__2Dread_8h_a08ba48255c6c28d067fa459419b9a18a}{Dataset\+\_\+2\+Dread}} body, because templates are instantiated at compile time. 

This function is used to read a dataset of dimension 1 or 2 from a $\ast$.h5 file.~\newline
\mbox{\hyperlink{Dataset__2Dread_8h_a08ba48255c6c28d067fa459419b9a18a}{Dataset\+\_\+2\+Dread}} reads the dataset given by the {\ttfamily dataset\+\_\+id} scanning it with memory slab whose dimensions are designed to be located inside dataset limit.~\newline
This is done because {\ttfamily H5\+D\+Read} is designed to crash if the user tries to read out of the dataset dimensions.

Memory slab read from the dataset is stored in a buffer {\ttfamily bufferdimension} $\ast${\ttfamily bufferdimension} (for 2D dataset or {\ttfamily bufferdimension$\ast$1} for 1D dataset) of type {\ttfamily buftype}.

Buffer is then written in the \char`\"{}output\char`\"{}.root file
